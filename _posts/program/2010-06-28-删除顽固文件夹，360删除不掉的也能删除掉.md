新建批处理文件，内容：
```bat
set R=%~1
if "%R:~-1%"=="." set R=%~1./
if "%R:~-1%"==" " set R=%~1./
echo y|cacls "%R%" /c /t /p Everyone:f>nul
taskkill /f /im explorer.exe>nul
Rd /s /q "%R%">nul
if exist %R% (for /f "delims=" %%b in ('echo %R%^|find /i "\com"') do rd /s /q \\.\%R%)
start "" "explorer.exe"
exit
```
使用方法，直接拖放顽固文件夹到这个批处理文件上即可。
